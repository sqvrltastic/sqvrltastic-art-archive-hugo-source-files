I use Adobe Photoshop (2019 version) as my drawing software of choice. By the way, I'd argue that, not only is it _not_ morally wrong, it is, in fact, every artist's moral imperative to pirate Adobe's Software!.. This is 100% a joke btw!

My graphic tablet is a decade old Wacom Cintiq Pro DTH-1320.

For 3D modeling I use Blender to make the model itself and then I later texture it in Substance Painter (circa 2018 version).

For coding and web development I use Visual Studio Code. All extensions that I use are listed in Coding resources links dump.
