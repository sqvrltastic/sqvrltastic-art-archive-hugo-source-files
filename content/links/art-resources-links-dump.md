# Art resources links dump

**Last updated:** January 30th 2025

_**Content warning** for some mild artistic nudity in some links!_

<a href="https://www.youtube.com/@ProkoTV" target="_blank" rel="noopener noreferrer">Proko YouTube channel</a> — feel like this is one of the best free resources out there for learning figure drawing

<a href="https://www.pureref.com/download.php" target="_blank" rel="noopener noreferrer">PureRef</a> — useful free little software for keeping and managing a reference board.

<a href="https://anatomy360.info/skull-sketcher-v2/" target="_blank" rel="noopener noreferrer">Skull Sketcher v2</a> — small program that lets you spin and light a 3D model of a skull, useful for anatomy drawing practice.

<a href="https://www.artstation.com/artwork/GX3Ax1" target="_blank" rel="noopener noreferrer">Male head 3D reference tool</a> and <a href="https://www.artstation.com/artwork/6a0Par" target="_blank" rel="noopener noreferrer">Female head 3D reference tool</a> by William Nguyen on ArtStation — again, another super useful drawing practice tool,

<a href="https://sketchfab.com/" target="_blank" rel="noopener noreferrer">Sketchfab</a> — this is a 3D model hosting website, but it's also a great way to look for various references that you can move and twist around.

<a href="https://colorhunt.co/" target="_blank" rel="noopener noreferrer">Color Hunt</a> — just some cool palettes for inspiration.
