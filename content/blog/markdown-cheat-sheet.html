Markdown cheat sheet

Markdown is a lightweight markup language that lets you add formatting elements to plaintext documents. It's used by a lot of SSGs (like mine, HUGO), but also in places like Discord and Reddit among others. Its main advantage is readability and simple syntax.

Different implementations of Markdown vary a bit between software and platforms. This guide is mostly from my own perspective and includes info about features that I personally use the most.

Basic text formatting

You separate paragraphs by adding an empty line in between them. You can also add a bit of basic formatting to your text, like shown below.

_italics_ or *italics*

**bold**

~strike-through~ or ~~strike-through~~

`code`

||spoiler|| (Discord only), or >!spoilers!< (Reddit only).

>blockquote

Headings

You should put a blank line before and after the heading. Also, examples below are not actual semantic HTML headers. It's just text formatted to look like headings that I'm using to illustrate my point.

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

Links

link

[link text](link URL)

Lists

Ordered list:

1. first
2. second
3. third

Unordered list:

- first
- second
- third

or:

* first
* second
* third

or, sometimes:

+ first
+ second
+ third

Other stuff

Horizontal rule (separator):

--- or ***
